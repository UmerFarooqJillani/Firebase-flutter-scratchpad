# <p align="center"> Connect Firebase Manual </p>

## PART C — Manual method
### Android manual
1. **Register app**
    - Use package name: `com.example.app`
2. **Download file**
    - `google-services.json`
    - Place here: `android/app/google-services.json`

### iOS manual
1. **Register app**
    - Use bundle ID.
2. **Download**
    - `GoogleService-Info.plist`
    - Add via Xcode.

### Why manual is not preferred
- error-prone
- platform-specific complexity
- hard to maintain

CLI handles all automatically.

# <p align="center"> Ways to connect Firebase </p>

## Connect Firebase to a Flutter app
**What this means**

You are linking:
> Your Flutter App ↔ Firebase Project (Backend)

Without this connection:
- Firebase cannot recognize your app
- your app cannot use Auth, Firestore, Storage

**Two methods**
1. FlutterFire CLI (Recommended) 
2. Manual setup (Android/iOS/Web config files)

Use CLI for real development.

# <p align="center"> Where data is stored </p>

## Auth
```
Firebase Authentication
```
**Use for:**
- login system
- identity (current user)

## Firestore
```
Cloud Firestore
```
**Use for:**
- user data, app data, structured data
- lists
- progress (real-time updates)

**Example:**
```
users/{uid}
```

## Storage
```
Firebase Storage
```
**Use for:**
- images
- files (file uploads)
- videos

**Example:**
```
profile_images/{uid}.jpg
```
